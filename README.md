# Three.js YouTube-Style Loader

Este proyecto es un ejemplo de un **loader animado en 3D** usando Three.js y Tween.js, inspirado en la animación de carga de YouTube. Incluye:

- Esferas girando en círculo con animaciones de posición, escala y color.
- Una barra de progreso 3D que aumenta su ancho de 0% a 100%.
- Texto 3D que muestra el porcentaje de progreso sobre la barra.
- Ajuste automático a la ventana (`onWindowResize`) para mantener la barra y las esferas visibles.

## Demo

Puedes ver el loader en acción directamente en tu navegador con Three.js y Tween.js.

## Tecnologías utilizadas

- [Three.js](https://threejs.org/) — Renderizado 3D en WebGL.
- [Tween.js](https://github.com/tweenjs/tween.js/) — Animaciones suaves para propiedades de objetos 3D.
- JavaScript (ES6 Modules).

> Nota: No requiere CSS adicional. Todo el renderizado y animación ocurre dentro de Three.js.

## Cómo usar

Clonar el repositorio o descargar el código y ejecutarlo con el interprete que sea.
